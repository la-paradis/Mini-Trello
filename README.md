# React + TypeScript + Vite

******************************************

### HTML-struktur (översikt)

Header (titel, användarinfo)

Main (boards-container)

Board

Kolumner

Kort

### TypeScript-funktioner (översikt)

Skapa / ta bort board

Lägga till / ta bort kolumn

Skapa / redigera / ta bort kort

Flytta kort mellan kolumner

Logga in / logga ut

Rendera innehåll till sidan

Kommunicera med backend (spara/hämta data)
